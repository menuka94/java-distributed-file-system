# CS555 - Distributed Systems
## Homework 1 - Distributed File System

### Version Info
* Java Version: 11
* Gradle Version: 5.2.1

### Build
* `make build`

### Run Controller (on `arkansas.cs.colostate.edu`)
* `make controller`

### Run Client
* `make client`

### Run ChunkServer (maximum 20 instances on 20 different hosts)
* `make chunk-server`
